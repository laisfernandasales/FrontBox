# 📦 Frontbox

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

Landing page de uma plataforma de cursos de programação, construída do zero com **HTML5 e CSS3 puro** — sem frameworks nem bibliotecas. O foco foi a estruturação semântica do HTML e boas práticas de estilização com CSS, incluindo custom properties, tipografia com Google Fonts e layout responsivo.

## 🚀 Como Usar

```bash
git clone https://github.com/laisfernandasales/FrontBox.git
cd frontbox
```

Abre o `index.html` no browser — ou usa o **Live Server** do VS Code para hot reload.

---

## 📄 Secções da Página

A página é composta por secções navegáveis via âncoras (`#somos`, `#cursos`, `#planos`, `#vantagens`):

**Menu → Header → Quem Somos → Cursos → Planos → Vantagens → Newsletter → Footer**
